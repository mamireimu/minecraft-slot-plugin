概要
このプラグインはチャット関連を使い、スロットの結果を表示するプラグインです。

仕様書
利用可能Ver: 1.11.2以降

/slot
	3列横、縦、対角線のうちどれかが揃えば以下の計算でインベントリーに入る
	ただし777の場合は、それだけ7倍する。
	鯖で設定した掛け金*(10*当たり数)+鯖で設定した掛け金*(1000*7のゾロ目だけの当たり数)

設定内容
・掛けるアイテムID
・掛けるアイテムの数
・貰えるアイテムID
・貰う際にブロック化する個数
・ブロック化したアイテム個数

アイテム
掛けるアイテム、貰えるアイテム、ブロック化したアイテム、いずれもサーバーのコンフィグで設定可能
貰えるアイテムは一定個数でブロック化してインベントリを節約するようにする。

なおアイテムIDは番号ではなく正式名で実行される。（今後のアップデートも含め）
